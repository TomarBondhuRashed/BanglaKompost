## Image Guidelines

- **Format**: JPG for photos, PNG for logos
- **Size**: Compress images for web (use TinyPNG.com)
- **Resolution**: 72 DPI minimum
- **Naming**: Use lowercase, hyphens, no spaces

## Free Stock Photo Resources

- [Unsplash](https://unsplash.com) - Search: compost, organic waste, farming
- [Pexels](https://pexels.com) - Search: sustainable, agriculture
- [Pixabay](https://pixabay.com) - Search: compost, recycling

## How to Add Images

1. Place images in this folder
2. Update `index.html` to reference them:
   ```html
   <img src="images/hero-bg.jpg" alt="Description" />
   ```
3. Or use as CSS background:
   ```css
   background-image: url("../images/hero-bg.jpg");
   ```
